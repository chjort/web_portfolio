# Web Portfolio Vision

A refined and minimalist architectural portfolio website targeting Scandinavian
architectural firms, particularly in Copenhagen. The website should communicate clarity,
analytical thinking, technical competence, and design maturity.

The website should showcase projects ranging from public spaces and urban interventions
to restoration and building design. The focus should be on demonstrating a strong
ability to synthesize complex information, develop rigorous design solutions, and carry
projects from concept through detailed design and implementation.

The website should feature:

* Clean, highly structured project presentations.
* Large visuals supported by drawings, sections, diagrams, and technical details.
* Clear documentation of design process and decision-making.
* A concise professional profile.
* Professional experience within a large architectural practice.
* Experience with permitting processes, regulatory compliance, and project coordination.
* Multilingual profile (English, Italian, Greek).
* Minimal animations and intuitive navigation.

The primary colours should be:

* Deep blue-green
* Soft white
* Light grey
* Graphite grey

The design philosophy is:

Quiet confidence. Simplicity. Architecture first.

The website should feel professional, precise, and internationally minded. It should
highlight both conceptual thinking and technical execution, presenting the architect as
someone capable of contributing to complex projects from early design stages through
detailed development and delivery.
